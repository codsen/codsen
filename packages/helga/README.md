## Install

This package is [pure ESM](https://gist.github.com/sindresorhus/a39789f98801d908bbc7ff3ecc99d99c). If you're not ready yet, install an older version of this program, 1.4.0 (`npm i helga@1.4.0`).

```bash
npm i helga
```
